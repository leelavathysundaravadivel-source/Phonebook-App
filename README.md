# Asynchronous REST Phonebook Directory Application

An active asynchronous mobile directory phonebook client dashboard engineered to consume standard server APIs via modern `async/await` architectural workflows. This single-page application manages data operations inside functional localized loops while checking formats prior to transit.

### Structural Performance Pillars Included
- **Asynchronous Data Layer Pipeline (2 Marks):** Implements network requests targeted to live placeholder APIs via native JavaScript `fetch()` syntax wrappers.
- **Full CRUD Entity Manipulation (3 Marks):** Handles retrieval tracking streams (`GET`), new contact uploads (`POST`), operational payload synchronization overlays (`PUT`), and secure directory record drops (`DELETE`).
- **Complete Error Resolution Matrix (1 Mark):** Detects connection disruptions, processing anomalies, and client-side status response thresholds, routing notifications through clear alerting modules.
- **Search Logic Core Filter (1 Mark):** Dynamically filters display nodes inside the active pipeline row elements without structural layout disruptions.
